# Statistical and decision rules

This project is a descriptive scoping evidence map and does not pool effect sizes.

## Inclusion access policy
Studies are included only when core charting fields are extractable from either (1) a sufficiently detailed abstract/record or (2) open-access full text. See `supplement/Appendix_2_Access_policy.md`.

## Follow-up reporting level
Downstream follow-up reporting is classified as:
- `none`: no mention of triage/referral/follow-up
- `referral_only`: referral/triage mentioned without completion or attendance metrics
- `completion_reported`: completion/attendance reported as a percentage or numerator/denominator

See `supplement/Appendix_3_Charting_codebook.md` for operational definitions.

## Rounding
Percentages are reported to one decimal place when shown alongside counts.
